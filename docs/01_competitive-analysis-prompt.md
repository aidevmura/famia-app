# ファミアアプリ 競合調査プロンプト
## ChatGPT / Gemini / Claude などへのインプット用

---

## このドキュメントの使い方

下記の「## アプリ概要」以降の内容を、ChatGPTやGeminiなどの生成AIに貼り付けて、以下のような質問をしてください。

**質問例:**
- 「このアプリの競合サービスを日本・海外問わず教えてください」
- 「このアプリが差別化できるポイントはどこですか？」
- 「このアプリにない競合の強い機能は何ですか？」
- 「このカテゴリのアプリで成功しているものの共通点は何ですか？」

## アプリ概要

### アプリ名
**ファミア（Famia）**

### コンセプト
家族みんなで使える、子どもの成長を楽しくサポートするファミリータスク管理アプリ。
ゲーミフィケーション（ポイント・アバター・アイテム）を活用し、子どもが自発的にお手伝いや習慣を継続できる仕組みを提供する。

### ターゲットユーザー
| ユーザー | 詳細 |
|---------|------|
| **主対象の子ども** | 3〜6歳（就学前〜低学年）|
| **保護者** | 父親・母親（20代〜40代） |
| **利用形態** | 家族単位での共同利用 |

### 解決したい課題
- 子どもにお手伝いや習慣を継続させるのが難しい
- 親が毎回「やったの？」と確認・催促しなければならない
- 子どもの小さな達成感を見える化・称える仕組みがない
- 家族全員がどこからでも同じ情報を確認できるアプリがない

---

## 主要機能

### 子ども向け機能
| 機能 | 詳細 |
|------|------|
| タスク一覧（今日のおしごと） | 毎日・毎週など設定された繰り返しタスクを表示 |
| タスク完了ボタン | タップひとつで完了報告、ポイント自動付与 |
| 「ひとりでできた！」ボタン | 自力達成時はボーナスポイントがもらえる |
| お祝いアニメーション | タスク完了時に紙吹雪・メッセージでモチベーションUP |
| アバターカスタマイズ | ポイントで買ったアイテムを装備してアバターを着替え |
| ショップ | ポイントを使って髪型・帽子・服・背景などを購入 |
| ピクチャーパスワード | 絵文字を選ぶだけの子ども向けログイン |

### 親向け機能
| 機能 | 詳細 |
|------|------|
| タスク管理（CRUD） | タスクの登録・編集・削除、カテゴリ・頻度・ポイント設定 |
| カテゴリ設定 | おそうじ・べんきょう・うんどう・たべる・その他 |
| 繰り返し設定 | 毎日・毎週・特定曜日 |
| 家族ダッシュボード | 家族全員の今日の進捗を一覧確認 |
| 家族管理 | 子どもアカウント確認、ファミリー招待コード発行 |

### 認証・アクセス
| 機能 | 詳細 |
|------|------|
| 親ログイン | メールアドレス＋パスワード |
| 子どもログイン | ファミリーコード＋ピクチャーパスワード（絵文字4つ選択） |
| PWA対応 | スマホのホーム画面にアイコン追加、アプリのように使える |
| マルチデバイス | iPhone・Android・PC ブラウザすべて対応 |

## 技術スタック（参考情報）
- **フロントエンド:** Next.js 14、TypeScript、Tailwind CSS、Framer Motion
- **バックエンド（予定）:** Supabase（PostgreSQL + リアルタイム同期）
- **デプロイ:** Vercel
- **現在のデータ永続化:** localStorage（デモ版）

## 競合調査で知りたいこと

生成AIへの質問テンプレート（コピーして使ってください）：

```
以下のアプリの競合サービスを調査してください。

【アプリ概要】
（上記の「アプリ概要」〜「主要機能」の内容を貼り付け）

【調査してほしい内容】
1. 日本国内・海外の競合アプリ一覧（アプリ名・URL・特徴）
2. 各競合との機能比較表
3. このアプリが差別化できる強みと弱み
4. 競合が持っていて、このアプリにない重要機能
5. このカテゴリで成功しているアプリの共通点・成功要因
6. ターゲット（3〜6歳の子ども向け）に特化した競合はあるか
7. マネタイズ方法の比較
```
